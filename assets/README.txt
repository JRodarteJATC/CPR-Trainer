LOGO / SEAL
===========

Save your Central Valley JATC seal in THIS folder as:

    logo.png

That's it. Once the file is named exactly "logo.png" and sits in this
assets/ folder, it automatically appears as:

  - the small emblem in the top bar
  - the faint watermark behind every screen
  - the watermark on the results popup

No code changes needed. If you ever want to use a different filename or a
.jpg/.svg, update the --logo line near the top of css/styles.css and the
<img> src in index.html.

Tip: a square image works best for the round emblem in the top bar (the
watermark looks fine either way).
